# Hospital Management System

A responsive Hospital Management System website developed using HTML, CSS, and JavaScript.

## Features
- Responsive UI Design
- Appointment Booking Form
- Doctor Profiles
- Hospital Services Section
- Testimonials Section
- Contact Information
- Form Validation using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Purpose
This project was developed to demonstrate frontend web development skills and responsive website design for healthcare management systems.
